# util-dfm

版权归统信软件技术有限公司所有
由libdfm-io 、libdfm-mount 和 libdfm-burn三个包组成的工具包

### 依赖
### 编译依赖

 debhelper (>=9),
 cmake,
 pkg-config,
 qtbase5-dev,
 qtbase5-private-dev,
 qt5-qmake,
 libglibmm-2.4-dev,
 qttools5-dev-tools,
 libglib2.0-dev,
 libudisks2-dev,
 libisoburn-dev,
 libmediainfo-dev,
 libsecret-1-dev
 

## 安装
### 源码安装

1. Build:

```bash
$ cd util-dfm
$ mkdir build
$ cd build
$ cmake ..
$ make
```

3. Install:

```bash
$ sudo make install
```
